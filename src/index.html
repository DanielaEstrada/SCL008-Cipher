CifrApp

La App que cifra

Funcionamiento

Esta App utiliza un método de cifrado por sustitución, es decir que cada letra del texto original es reemplazada por otra que se encuentra un número fijo de posiciones (desplazamiento) más adelante en el mismo alfabeto. Puedes encontrar más información AQUÍ.

Podrás crear fácilmente contraseñas y mensajes cifrados.

Para comenzar, primero lee las instrucciones de uso.

Instrucciones de uso

Para crear contraseñas y/o un texto cifrado primero debes elegir un número secreto mayor que cero (recordar para después descifrar), luego escribir lo que quieres cifrar/descifrar, y presionar el botón "cifrar" o "descifrar" según corresponda.

  • Puedes utilizar letras, números y símbolos.
  • Utiliza sólo letras del alfabeto simple (sin ñ).
